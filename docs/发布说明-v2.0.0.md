# OpenClaw 一键部署 v2.0.0

本次版本重点完善了 Windows 首发可用性，补齐了一键安装、环境检测、内置资源安装、多渠道接入和常见问题处理流程，适合先快速完成 OpenClaw 部署，再逐步接入 Telegram、QQ、飞书、Discord、钉钉等渠道。

## 本次更新

- 新增 Windows 图形化部署流程，降低 OpenClaw 首次安装和配置门槛
- 优化环境检测与修复提示，补充 Rust、Build Tools、WebView2 等依赖说明
- 打包内置 OpenClaw 运行资源和常用渠道扩展，适合受限网络环境下完成安装准备
- 调整首页和界面文案，更适合首发公开发布
- 补充安装包、绿色版、Shell 脚本等多种使用方式

## 下载方式

- Windows 安装版：前往 GitHub Releases 下载最新 `.exe` 安装包
- Windows 绿色版：下载 `OpenClaw-Deploy-v2.0.0-Windows.zip` 后解压运行

## Linux / macOS 一键脚本

直接安装：

```bash
curl -fsSL https://raw.githubusercontent.com/3445286649/openclaw-deploy/main/install.sh | bash
```

交互式菜单：

```bash
curl -fsSL https://raw.githubusercontent.com/3445286649/openclaw-deploy/main/install.sh | bash -s menu
```

当前版本更聚焦部署与接入体验，适合作为 OpenClaw 的首发桌面安装工具使用。
